# 77. Combinations

**Source:** [https://leetcode.ca/all/77.html](https://leetcode.ca/all/77.html)

**Companies:** Adobe, Amazon, Apple, Facebook, Google, Microsoft

## Description

Given two integers

n

and

k

, return all possible combinations of

k

numbers out of 1 ...

n

.

## Examples

### Example

```
Example:
Input:
n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```
