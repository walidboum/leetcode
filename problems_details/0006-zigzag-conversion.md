## Description

The string

"PAYPALISHIRING"

is written in a zigzag pattern on a given
        number of rows like this: (you may want to display this pattern in a fixed font for better
        legibility)

A P L S I I G
Y   I   R

And then read line by line:

"PAHNAPLSIIGYIR"

string convert(string s, int numRows);
