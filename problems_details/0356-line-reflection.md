# 356. Line Reflection

**Source:** [https://leetcode.ca/all/356.html](https://leetcode.ca/all/356.html)

**Companies:** Amazon, Google

## Description

Given n points on a 2D plane, find if there is such a line parallel to y-axis that reflect
        the given points.

Example 1:

Input:

[[1,1],[-1,1]]

Output:

true

Example 2:

Input:

[[1,1],[-1,-1]]

Output:

false
