# 1118. Number of Days in a Month

**Source:** [https://leetcode.ca/all/1118.html](https://leetcode.ca/all/1118.html)

**Companies:** Amazon

## Description

Given a year

Y

and a month

M

, return how many days there are in
        that month.

Example 1:

Input:

Y =

1992

, M =

7

Output:

31

Example 2:

Input:

Y =

2000

, M =

2

Output:

29

Example 3:

Input:

Y =

1900

, M =

2

Output:

28

Note:

1583 <= Y <= 2100

1 <= M <= 12
